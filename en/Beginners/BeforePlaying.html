    注意检查游戏更新

    双击灵梦图标的 SokuLauncher.exe 启动游戏，每次打开时会自动检测现有 mod 有无更新

    • 非想天则资源下载

    • SokuLauncher 更新器使用说明

    • 注意检查 游戏标题的 giuroll 版本，以及 大厅界面右下角的大厅版本

    • 如果全屏看不到游戏标题，按 Alt + Enter 即可切换全屏和窗口化

    开始游戏前，设置好你的机签名（Profile）、键位、卡组

    Info

    • 若要修改机签名，请打开你的 th123 游戏文件夹，找到 profile 文件夹，进入后将 profile1p.pf 修改为你想要的文件名即可，如修改为 则则新人旗美铃.pf，修改后再重新进入 C键 菜单。

    • 先选择好机签，右上角变化后，再去设置 1P 卡组和按键

    • 机签名字最好是可以方便在 QQ 等地方找到你的名字，不要设置低俗无礼或意义不明、不好打出来的名字

    • 可以将自己的游戏水平标注，有助于寻找旗鼓相当的对手

    确保网络连接稳定

    Caution

    • 显卡驱动有问题可能会导致游戏卡顿
    • 低电量（节能模式）或高占用 CPU 内存可能会导致游戏卡顿
    • 为什么联机很卡

    检查网络波动情况

    按下 Win+R，粘贴 cmd /k ping baidu.com -t 后回车，稍微等待一会，
    延迟有上下 10ms 左右的波动都属于正常情况。

    也可以检查游戏对局内 正中央下方数字网络延迟 有没有一直在跳变，若有则很可能说明有人是 Wifi，网络不稳定，不建议进行游戏

    手机流量如何USB共享给电脑

    1. 手机关掉 Wifi，打开数据流量，USB连接电脑，电脑也关掉 Wifi，注意提前关掉其他可能会消耗流量偷跑流量的东西（如迅雷、浏览器后台视频等）

    5. 然后右下角网络图标应该是这样 ，就可以了

    使用 Swarm 等中转 IP 在大厅里联机

    联机教程 - Swarm使用说明

    通过中转服务器，中转数据包的方式来帮助联机，只需要建主方打开 Swarm 软件即可，对连接方无要求

    1. 点“获取服务器列表”（一般选择自己延迟最低的就行，绕远路时综合双方的列表进行选择）

    2. 点“中转”可以得到一串 IP，然后点“复制”就自动帮你复制好了，可以发给别人了

    3. 如果别人进不来，请尝试再次点击“中转”，然后“复制”新的 IP 发给别人

    4. 如果你想在大厅建主等待别人加入，可以点击“IP 用于大厅”的按钮；
      重新点了“中转”刷新 IP 的话，也记得要再次点击“IP 用于大厅”，不需要重开游戏或重进大厅

    • 中转后点“端口管理”，可以查看自己和对手的网络延迟情况
    • 右下角图标是“黑名单”功能

    提醒注意使用 swarm 不要绕远路了

    正好处于下面所写四个省份的可以无脑选自己所在省份，其他的人一般选用离得近的地区作为中转

    如果双方网络延迟过大，很有可能是绕了远路，此时联机双方两个人，可以都截图发下自己的列表延迟，然后综合选择相加延迟最小的的中转服务器

    如果支持 AutoPunch 直连 或者 IPV6 直连，那是最好不过了

    服务器所在地区：

    • 124.223.180.38 —— 上海（东部）
    • 8.134.214.80 —— 广东广州（南部）
    • 47.108.225.175 —— 四川成都（西部）
    • 152.136.215.92 和 101.43.209.105 —— 北京（北部）

    实际延迟 ≈ 主机到服务器的距离 + 客机到服务器的距离

    绕远路错误示范和顺路正确示范，内多图，手动点击放大查看

    Swarm被误杀，刷不出服务器

    Swarm 可能会被杀毒软件误报误删，请添加信任或白名单并从恢复区恢复

    如果遇到点 “获取服务器列表” 但是刷不出 IP 的话，请检查 Windows防火墙 和 杀软误杀

    实在不行或者赶时间的话，也有备用联机工具

    1、不会被杀软误杀的旧版本 Swarm

    如果不方便解决的杀软误报等问题而无法使用新版 Swarm 的话，也可以使用旧版本 Swarm，带猫头鹰图标的那个，一般整合包内已经包含有 Swarm-old 了，不必再次下载

    2、DAxZE by 喵音帝 YIU

    DAxZE by 喵音帝YIU，通过 萌卡(mycard) 的服务器进行中转，和 Swarm 功能类似，在整合包的 th123 文件夹内可以找到

    详细请看其Lofter页面open in new window | 详细请看其Github页面open in new window

    　

    进入 VS NETWORK 菜单，开始联机

    Important

    • 按键盘 S键 可以快速切换成原版传统菜单

    • VS NETWORK 菜单界面操作见下方图片 👇

    联机方式

    自己建主，做1P

    • 如果你是用大厅：在大厅里没人的机子上按确认键 Z

    • 如果你是用QQ群：在传统原版菜单，选择第一项 建立主机，端口 10800 不用修改，然后看到等待连接中就可以发 IP 出去了

    • 若是自己建主，向群里发IP，最好附带相关信息，如：

    42.193.7.216:58309 对4 妖梦 （不带信息，谁都能进）
    42.193.7.216:58309 对5 UU，等到12点/打十局/抢十 （注意别人已经说好了打多久）
    42.193.7.216:58309 新人对三灵梦，希望水平差不多的其他新人 （意思老手不要进来）
    42.193.7.216:58309 3+文，希望水平高点的指教 （意思新手不要进来）
    42.193.7.216:58309 @xxx （意思是有约了，别的人不要进来）
    42.193.7.216:58309 对二 黑白 妖梦，随时切 （意思是可能随时会中断游戏）

    自己进别人，做2P

    • 如果你是用大厅：在有一个人建主的机子上按确认键 Z，就会加入了

    • 如果你是用QQ群：在传统原版菜单，复制别人的 IP，选择 从剪贴板复制的IP加入 即可

    不要用下图那种密码锁形式输入 IP

    若电脑上没有登陆QQ，可以随便在电脑上什么地方（如左下角搜索栏）敲键盘打字，打出 IP 来，然后再复制会方便很多

    成功进入对战，选人界面

    成功进入对战，选人界面

    • 进入后，请在QQ群内回复别人，让别人知道是在和谁玩，便于后续交流

    • 可在选人界面按C，能且只能修改卡组

    • 若不想继续打了，可以在群里at对手，或者在选人界面连续按X发出框框的声音，表示Last，最后一局

    • 如果发现对手一直没反应，最好查看下QQ群消息和大厅聊天框，也许他正在那边打字

    注意正中央网络延迟并调整左下角延迟帧数

    进入对局后

    • 如果发现对手一直没反应，请理解并耐心等待，他可能身边有点急事需要离开一会儿；可以查看下QQ群消息和大厅聊天框，也许他正在那边打字

    • 如果发现自己和对手实力悬殊（对手太强或者太新），可以在QQ群或聊天框内交流是否还要继续玩

    • 最好不要无端故意中途ESC结束对局，会导致无法保存Rep，若有紧急情况要离开可以在事后QQ群或者聊天框通知对手

    • 2023.11.30 Hagb制作了SaveRep的mod可以在异常情况下紧急保存rep，虽然ESC也可以保存Rep了但是依然不提倡“节省时间”而在对局时ESC

    • 打完结束后可以按两下X框框两声，并回复“谢谢指教”，“谢则”之类的话感谢对战

    注意正中央网络延迟并调整左下角延迟帧数

    使用 大键盘的加减号(在 Backspace 键的左边) 调整左下角 delay 值，调整规则可参考下图

    简单来说就是：从d1开始，若发生频繁回滚现象则增大输入延迟，直到均衡满意

    Giuroll 0.6 系列版本，增加“单方回滚”特性：对战双方各自按大键盘加减号（Backspace退格键左边的那个），来调整自己的左下角延迟帧数，互相不影响对方

    延迟数值与左下角输入延迟之间关系的详细解释

    假设游戏内正下方的数字显示 40，说明双方网络延迟是 80ms 左右，就相当于总共延迟 5 帧，此时

    1P 设置左下角 d1，就选择延迟 1F（1P 的指令输入像练习模式一样流畅），但是就需要 4F 的回滚帧（1P 画面里的对手发生回溯与瞬移的情况频发）；

    2P 设置左下角 d3，就选择延迟 3F（2P 输入的指令过 3F 才会实际输入游戏内），不过只需要 2F 的回滚帧（2P 画面里的对手基本不会发生回溯与瞬移）
